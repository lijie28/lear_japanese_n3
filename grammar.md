#**<font color=blue>1日目　ぼくにもやらせて</font>**

「怪我をされると困るからダメよ。」

#Vられる＜受身形＞
##主語を表さないで事実を述べるとき、よく受身形を使う
###例文：
この本<font color=red>には</font>、詳しい説明<font color=red>は</font><u>書かれていません</u>。 （＝＜筆者が＞書いていない）
入学式は、このホールで<u>行われます</u>。（＝＜学校が＞行う）
これは、世界で一番大きいタイヤモンド<font color=red>だ</font>と<u>言われています</u>。（＝＜誰かが＞言っている）
昔は、その考え<font color=red>が</font>正しい<u>と思われていた</u>。（＝＜昔の人は＞思っていた）
###練習：
英語は世界で<u>話されています</u>。
この建物は三百年前に<u>建てられました</u>。
オリンピックは4年に一度<u>開かれます</u>。

# （Nに）Vられる＜受身形＞
##そのことによって、困る状態になったことを表す。
```
Vられてしまった
```
###例文：
友達の赤ちゃんを抱っこしたら、<u>泣かれてしまった</u>。
雨<font color=red>に</font>降られて、服<font color=red>が</font><u>濡れてしまった</u>。

###練習：
急に社員<font color=red>に</font><u>辞められた</u>

#V（さ）せて - ください／もらえますか／もらえませんか
##自分の行動について、相手から許可を得る表現。
###例文：
ちょっと気分を悪いので、早く<u>帰らせてください</u>。
あなたの会社のお話<font color=red>を</font><u>聞かせてください</u>。
手を<u>洗わせてください</u>。
###練習：

#Vないと／Vなくちゃ
##「Vないといけない」（＝Vなければならない）を省略した言い方。
###例文：

#V~~て~~ちゃう
```
Vで→Vじゃう
```
##『Vてしまう』の会話的表現。

```
してしまう→しちゃう
来てしまう→来ちゃう
帰ってしまう→帰っちゃう
死んでしまう→死んじゃう
```
###例文：
- 『あれ？ここにあったチョコレートは？』
『あ、<u>食べちゃった</u>。いけなかった？』（＝食べてしまった）
- 試験<font color=red>が</font>1終わった！今日は<u>飲んじゃおう</u>！（＝飲んでしまおう）
- もしもし、すみません。車<font color=red>が</font>1<u>混んじゃって</u>〜少し遅れます。（＝混んでしまって）
###練習
- あ、バス、行っちゃった。
- あーあ、雨に降られちゃった。
- 起きないとと思ったのにまた寝ちゃった。
- 朝早く起きて作っといたお弁当<font color=red>を</font>家に置いて来ちゃった。
 
#V~~て~~とく
```
Vておく→Vとく
Vでおく→Vどく
Vておかない→Vとかない
```
##『Vておく』の会話的表現
```
しておきます→しときます
見ておいた→見といた
読んでおきます→読んどきます
飲んでおいて→飲んどいて
```
###例文：
- テストで間違ったところを、ノート<font color=red>に</font>1<u>書いとこう</u>。（＝書いておこう）
- これ<u>洗濯しといて</u>。（＝洗濯しておいて）
- <u>買っとかないと</u>。（＝買っておかないと＜いけない＞/買っておかなければならない）
###練習
- 「パソコン消す？」『まだ使うからつけといて。』
- これ、来週まで<font color=red>に</font>読んどいて。

#N/V/A/na/N普   -   みたいだ。／みたいに／みたいなN
##「〜ようだ」よりも話しことば的。
###例文：
- 彼の話し方は、<u>女みたいだ</u>。（＝女のようだ）
- ここの砂は<u>星みたい<font color=red>な</font></u>形<font color=red>をしている</font>。（＝星のような）
- このアパートは誰も<u>住んでいないみたいだ</u>。（＝住んでいないようだ）
- 明日は<u>雨みたい</u>ね。（＝雨のようだ）
- <u>リンさんみたい<font color=red>に</font></u>日本語がうまくなりたい。（＝りんさんのように）
###練習：
- 冬なのに今日は暖かくて春みたいです。
- 試験を受けたけど、合格は無理みたいです。
- 今からお話しするのは、嘘みたいな本当の話です。

#Nらしい
```
子供らしい
男らしい／女らしい
私らしい／君らしい
```
###例文：
今日は、<u>春らしい</u>暖かい日でした。（＝本当に春だと感じる）
私<font color=red>は</font><u>女性らしい</u>洋服<font color=red>は</font>あまり着ない。（＝みんなが女性の服だと感じる）
###練習：
- うちの子<font color=red>には</font>子どもらしく元気<font color=red>に</font>育ってほしい。
- 明日はよく晴れて秋らしい天気になるでしょう。

#Nっぽい
```

```

###例文：
あの小学生は、<u>大人っぽい</u>。（＝大人みたいだ）

###練習：
- 母はいつも子供っぽい服を着ている。
- 逃げた<font color=red>のは</font>黒っぽい車でした。

#**<font color=blue>4日目　できるようになった！</font>**
<font color=blown>

「手を切らないように気をつけて！」

#Vる／Vない　−　ようにする
##習慣、心がけ（努力）を表す。
###例文：
- 忘れ物を<u>しないようにしましょう</u>。（＝忘れ物をしない努力をしましょう）
- 毎食後、<u>歯を磨くようにしています</u>。（＝歯を磨く努力をしています）
###練習：
- 健康<font color=red>のために</font>野菜を食べる<font color=red>ように</font>しています。
- インフルエンザ<font color=red>に</font>ならない<font color=red>ように</font>うがいや手洗いをする<font color=red>ように</font>しています。


#Vる＜無意識動詞＞／Vない／Vれる　−　ように
#Vる＜意識動詞＞　−　為に
###例文：
- みんなに<u>聞こえるように</u>もっと大きな声で話してください。（＝聞こえないと困るから）
- <u>忘れないように</u>、手帳に書いておこう。（＝忘れる突困るから）

- `ダメ`　~~家を買うように貯金をしている。~~
→家を<u>買う為に</u>貯金をしている。（＝買うことが目的で）
- `ダメ`　~~働くように学校を辞めた。~~
→<u>働く為に</u>学校を辞めた。（＝働くことが目的で）
###練習：
- 風<font color=red>が</font>入らないように窓を閉めておきましょう。
- 大学<font color=red>に</font>合格できるように一生懸命勉強しています。

#Vるようになる
###例文：
- 「このDVDプレーヤー、壊れているんだよね。」
「いや、直してもらったから、<u>使えるようになった</u>よ。」（＝使える状態になった）
- ここの電気<font color=red>は</font>人<font color=red>が</font>通る<font color=red>と</font><u>つくようになっています</u>。（＝つく状態になっている）

###練習：
- 日本人みたい<font color=red>に</font>日本語<font color=red>が話せる</font>ようになりたい。
- このインターホンは留守<font color=red>に</font>誰が来たかわかるようになっている。
- たいていの電気製品は一年以内なら壊れてもタダで直してもらえるようになっている。

#Nの／V　−　ように
##説明の前置きでよく使う。
```

田中さんが言ったように
```
###例文：
- 皆様<u>ご存知のように</u>、試験の内容<font color=red>が</font>変わります。（＝知っている内容と同じで）
- ここに<u>書いてあるように</u>、今度の木曜日、授業は午前中だけです。（＝書いてある内容と同じで）
- <u>前にお話ししたように</u>、来年から授業料<font color=red>が</font>値上がりします。（前に話した内容と同じで）
###練習：
- 次のように修正してください。
- 今、テレビなどでコマーシャル<font color=red>を</font>しているように、これの新製品<font color=red>が</font>もうすぐ発売されます。


#Vる／Vない　−　ように。
##軽い命令を表す
```
するようにしましょう
するようにしなさい
```
###例文：
- 「明日はもっと早く<u>来るように</u>。」（＝来なさい）
「わかりました。」
- ここで、タバコを<u>吸わないように</u>。（＝吸わないでください）
###練習：
- 遅れる場合は連絡するように。
- 雪が降っていますから、転ばないように注意して出かけてください。
- いつも言っているように無理を<font color=red>しすぎない</font>ように。

#Vます／Vません／Vれます　−　ように。
##願望を表す。
```
勝ちますように
勝てますように
負けませんように
```
###例文：
- <u>合格し<font color=red>ます</font>ように</u>。（＝合格することを望んでいます）
- <u>母の病気が治り<font color=red>ます</font>ように</u>。（＝治ることを望んでいます）
- インフルエンザ<font color=red>に</font><u>かかり<font color=red>ません</font>ように</u>。（＝かからないことを望んでいます）
###練習：
- いい仕事<font color=red>が</font>見つかり<font color=red>ます</font>ように
- 私が作った曲を皆さん<font color=red>が</font>喜んでくれ<font color=red>ます</font>ように。

#Vようと思う
##意志や予定を表す。
###例文：
- 会社を<u>辞めようと思っている</u>。（＝辞めるつもりです）
- 来年、国に<u>帰ろうと思っています</u>。（＝帰るつもりです）
###練習：
- 「昨日は授業に来なかったね。」
「うん、来ようと思っていたんだけど、熱が出て休んだんだ。」
- 買おうと思って書い<font color=red>といた</font>メモを忘れてきちゃった。
- 弟の好きなCD<font color=red>を</font>誕生日<font color=red>に</font>買ってやろうと思っています。

#Vようとする
###例文：
- 電車に<u>乗ろうとした時に</u>、ドアが閉まって乗れなかった。（＝乗る動作を始めた時に）

- 彼<font color=red>は</font>、自分のこと<font color=red>は</font>何も<u>言おうとしない</u>。（＝言うつもりがない）
###練習：

- このジャムのふたは、いくら開けようとしても開かない。
